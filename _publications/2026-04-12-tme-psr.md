Sequential recommendation systems aim to predict user preferences based on historical interactions, but often neglect the temporal dynamics, diverse user interests, and the need for explainability. We propose TME-PSR, a unified framework that jointly addresses three critical challenges: (1) **Time-awareness** — modeling how user preferences evolve over time through temporal encoding and decay mechanisms; (2) **Multi-interest** — capturing diverse user interests via a multi-head attention mechanism that disentangles distinct preference patterns; and (3) **Explanation personalization** — generating tailored natural language explanations for recommendations using a personalized explanation module. Extensive experiments on real-world datasets demonstrate that TME-PSR achieves state-of-the-art recommendation accuracy while providing meaningful, personalized explanations that enhance user trust and satisfaction.
